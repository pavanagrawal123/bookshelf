    What is this software? Click blue buttons for answers.

    This project is supposed to keep track of books for Ms. Smith's classroom. Each book has a set amount of ebooks that Ms. Smith can distribute, but each ebook can only be redeemed once. To help keep track of this, Ms. Smith is using this software to mark down which students have which ebooks checked out. Using bookshelf (the name of this software), Ms. Smith is able to generate reports on how many book she has, who has checked out books, and how claim codes are distributed.
